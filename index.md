## 🚀 Getting Started

Welcome to the Arduino Stress Monitor! This simple tool helps you measure your heart rate in real time and classify your stress levels as low, normal, or high. Follow these steps to download and run the software easily.

## ⚙️ How to Run the Software

Once you have downloaded the software, follow these steps to run it:

1. **Locate the Downloaded File**:
   - Open your Downloads folder.
   - Find the downloaded file named `arduino-stress-monitor.zip` or similar.

2. **Extract the Files**:
   - Right-click on the downloaded file and select "Extract All" (Windows) or "Open" (macOS).
   - Choose a destination folder or use the default folder.

3. **Open the Application**:
   - Navigate to the extracted folder.
   - Locate `arduino-stress-monitor.exe` (Windows), `arduino-stress-monitor.app` (macOS), or a similar file.
   - Double-click the file to open the application.

## 📊 Using the Application

Now that the application is running, you can start measuring your heart rate:

1. **Connect the Device**:
   - Ensure your Arduino board is connected to your computer via USB.
   - Use the appropriate heart rate sensor as per the guidelines provided in the device documentation.

2. **Start Measuring**:
   - Click the “Start” button in the application to begin monitoring your heart rate.
   - The application will display your current heart rate and stress level.

3. **Interpreting the Results**:
   - **Low Stress**: Heart rate below 60 BPM.
   - **Normal Stress**: Heart rate between 60-100 BPM.
   - **High Stress**: Heart rate above 100 BPM.

## ❓ Troubleshooting

If you encounter issues while running the software, here are some common solutions:

- **Connection Problems**:
  - Make sure the Arduino board is properly connected.
  - Check if the correct port is selected in the application settings.

- **File Not Opening**:
  - Ensure you have extracted the files correctly.
  - Verify that your operating system supports the file format.

- **Data Not Updating**:
  - Restart the application.
  - Check the sensor connection and ensure it is working correctly.
